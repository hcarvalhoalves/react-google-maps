react-google-maps

View project on GitHub

Google Maps controls wrapped into reusable React components.

Available components

GMaps.StaticMapWidget

Supported props

latitude
Latitude in decimal.
longitude
Longitude in decimal.
width
Width in pixels. Limited to 640 (2048 for Google Maps API for Business).
height
Height in pixels. Limited to 640 (2048 for Google Maps API for Business).
scale
Resolution factor. Accepted values are 1 or 2 (4 for Google Maps API for Business).

GMaps.MapWidget

Supported props

latitude
Latitude in decimal.
longitude
Longitude in decimal.
zoom
Initial zoom level. Default to 4.
width
Width in pixels.
height
Height in pixels.
draggable
Can click and drag the map. Default to false.
scrollwheel
Can zoom with mouse wheel. Default to false.
zoomControl
Show +/- buttons. Default to true.
scaleControl
Show scale slider. Default to false.
panControl
Show panning arrows. Default to false.
mapTypeControl
Allow selecting Map/Terrain/Satelite views. Default to false.
streetViewControl
Allow Street View. Default to false.
overviewMapControl
Toggle overview map controls. Default to false.
points
Add markers to the map. Should be an array of objects with the following attributes:
latitude
Latitude in decimal.
longitude
Longitude in decimal.
title
Title shown while hovering this marker.
icon
Absolute URL to custom icon. Optional.
onClick
Callback to trigger when this marker is clicked.

You can find more information at the Google Maps API official docs.

GMaps.AutocompleteField

Supported props

types
An array of supported lookup types. Defaults to ['geocode']. See the official documentation for other options
country
Limit results to suggestions inside a country. Must be passed as as a two-character, ISO 3166-1 Alpha-2 compatible country code. Optional.
bounds
Limit results to a bounding box. Passed as a pair of coordinates in the format [[lat, lon], [lat, lon]].
onPlaceChanged
Callback called when the user selects one or zero results. Receives two arguments:
latlon
Coordinates of the result in the format [lat, lon].
distance
Radius of the result, if applicable. Useful to set the zoom level of a map control and have the entire result in view.

Important: To use this component is necessary to include the Google Maps API client with libraries=places appended to the query string.
